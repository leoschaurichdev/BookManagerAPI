# BookManager

## Descrição
Este é um projeto WEB API para gerenciar o empréstimo de livros em uma biblioteca.

## Recursos Utilizados
- Visual Studio 2022
- Framework .Net 8.0 e linguagem C#
- ORM Entity Framework Core
- Implementação do padrão CQRS utilizando a lib MediatR
- Swagger
- SQL Server
